# Changelog

## [0.1.0] — 2026-06-29 🎉

### 项目初始化
- **架构搭建**：808 设备网关 (Node.js + TypeScript) + 管理后台 (React + Ant Design) + 微信小程序
- **协议实现**：JT/T808 帧解析骨架（注册、鉴权、心跳、位置汇报等核心消息类型）
- **部署配置**：Docker Compose 一键部署

### 核心功能
- 设备绑定与管理（用户-设备关联、在线/离线监控）
- 远程控制（断电、通电、锁车、解锁）
- 实时定位（GPS/北斗/Wi-Fi/LBS 多重定位）
- 电子围栏（圆形/矩形区域、进出检测）
- 异常告警（超速、震动、主电断开、非法拆除）

### 🧹 去商业化改造（2026-06-29）
- **移除 VIP 会员体系**：删除 VIP 中心、会员费、续费逻辑
- **移除分销/佣金系统**：删除邀请奖励、回馈金、设备免单
- **移除商家/合作商体系**：删除商家申请、合作商钱包、提现审核
- **移除钱包/支付**：删除钱包、充值、订单支付相关功能
- **商城→兼容硬件**：将商城重新定位为"兼容硬件参考"，无价格/交易功能
- 清理管理后台商业相关页面（运营、渠道、提现审核等）
- 清理后端商业逻辑（VIP 权限、优惠计算等）
