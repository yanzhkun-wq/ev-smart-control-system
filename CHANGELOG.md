# Changelog

## [0.1.0] — 2026-06-29 🎉

### 项目初始化
- **架构搭建**：808 设备网关 (Node.js + TypeScript) + 管理后台 (React + Ant Design) + 微信小程序
- **协议实现**：JT/T808 帧解析骨架（注册、鉴权、心跳、位置汇报等核心消息类型）
- **部署配置**：Docker Compose 一键部署

### 核心功能
- 设备绑定与管理（用户-设备关联、在线/离线监控）
- 远程控制（断电、通电、锁车、解锁）
- 实时定位（GPS/北斗/Wi-Fi/LBS 多重定位）
- 电子围栏（圆形/矩形区域、进出检测）
- 异常告警（超速、震动、主电断开、非法拆除）
