## 🔑 核心技术方案

### 协议层 (JT/T 808)
- 基于国标 JT/T 808 V2.8（WZ 版）
- 0x7e 帧定界 → 转义还原 (0x7d 0x01/02) → 消息头/体解析
- 校验码：消息头第一字节至校验码前一字节异或
- 支持分包重组（消息体属性 bit13）

### 设备网关
- TCP 长连接，独立 Socket 会话管理
- 单终端连接策略（防重复上线）
- 厂家适配器模式（默认 JT808 → 可扩展私有协议）
- 消息路由：按消息 ID 分发

### 数据存储
- JSON 文件持久化（当前）
- 轨迹：按终端按日存储为 JSONL 文件
- 规划：Redis（实时缓存）+ PostgreSQL（历史存储）
