# WZ 通用版部标 JT/T808 协议 — 摘要（V2.8）

> **原文 PDF**：与本摘要同目录的 `WZ标准版808协议V28.pdf`（文档内版本号为 **V2.8**，2025-04-01 起含无线设备开关机报警位等）。  
> 实现以 PDF 为准；本文件便于检索与写代码时对照。

## 通信与帧格式

- **链路**：TCP；**平台为服务端**，终端为客户端（JT/T 794 相关约定）。
- **字节序**：大端；校验码：从消息头第一字节起至校验码前一字节，逐字节异或。
- **标识位**：`0x7e`；转义：消息中出现 `0x7e` → `0x7d 0x02`；出现 `0x7d` → `0x7d 0x01`。
- **消息头**：消息 ID(WORD)、消息体属性(WORD)、终端手机号 BCD[6]、流水号(WORD)；分包时 bit13=1，后跟总包数/包序号。

## 0x0200 位置基本信息（28 字节起 + 附加项）

- 报警标志 DWORD、状态 DWORD、纬度/经度 DWORD（度×1e6）、高程 WORD、速度 WORD（**1/10 km/h**）、方向 WORD、时间 BCD[6]（GMT+8）。

### 状态位（表 17，摘录）

| 位 | 含义 |
|----|------|
| 0 | ACC 关/开 |
| 1 | 未定位/定位 |
| 2–3 | 纬/经南北东西 |
| 6 | **撤防/设防** |
| 10 | 油路正常/断开 |
| 11 | 主电正常/**主电断开** |
| 13 | 车门关/开 |
| 18 | GPS 定位标志 |
| 19 | 北斗定位标志 |

### 报警标志（表 18，摘录）

含：紧急、**超速**、疲劳、主电欠压、**主电源断开**、无线低电、**震动**、**拆除**、超时停车、非法位移、车门开、**ACC 开/关报警**、设备开/关机等（详见 PDF；部分位“收到应答后清零”）。

## 客户 808 平台兼容配置（第 3 章）

设备可通过参数映射**报警位、状态位、扩展字段、基站/Wi‑Fi 开关、2011/2013 选择**等（如 `808_SFFLAG`、`VIB_ALARM_808`、`808_LBS`、`WIFI_ENABLE`、`808_SEL` 等）。平台侧应支持按终端/客户记录这些差异或依赖终端出厂配置。

## 串口透传（2.10）

- 下行最大 512 字节（含 1 字节 808 串口 ID，数据最大 511）；上行最大 1024 字节。
- 上行 `0x0900` 首字节固定 `0x41`（串口 1）；下行 `0x8900` 首字节由终端过滤。
